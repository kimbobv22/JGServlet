requestURL

(
  • urlKey_
  • [objects_]
)
String

Defined in JGService.3.0.0.js:20

요청URL을 설정/반환합니다.

  1. 두번째 매개변수가 객체일 경우, 매개변수와 함께 GET방식의 URL로 반환합니다.
  2. 두번째 매개변수가 문자열일 경우 해당 문자열을 요청URL로 설정합니다.

Parameters:

  • urlKey_ String

    URL키

  • [objects_] Object optional

    JSON형식의 매개변수 또는 설정할 요청URL

Returns:

String:

요청URL

Example:

//요청 URL 설정하기
JGService.requestURL("key", "http://localhost/context/{mData1}");

//요청 URL 반환하기(매개변수 형식)
var requestURL_ = JGService.requestURL("key",{
    appendPath : "sample", //경로확장
    parameters : {param1 : "abc"}, //GET파리미터
    mappingData : {mData1 : "xyz"} //매핑데이터
});
//결과값
// http://localhost/context/xyz/sample?param1=abc

sendMultipart

(
  • urlKey_
  • [urlData_]
  • [ajaxOptions_]
  • [inputFile_]
)

Defined in JGService.3.0.0.js:216

요청URL로 매개변수와 함께 Multi-part 요청을 수행합니다.
inputFile_ 매개변수가 존재하지 않을 경우 파일선택창을 호출하여 요청을 진행합니다.

옵션항목은 jQuery AJAX Options가 동일하며 기타 옵션은 아래와 같습니다. //기타옵션항목 { ,multiple : "파일다중선택여부" ,accept : "허용파일항목" };

Parameters:

  • urlKey_ String

    URL키

  • [urlData_] Object optional

    JSON형식의 매개변수

  • [ajaxOptions_] Object optional

    jQuery AJAX 옵션

  • [inputFile_] JQuery Selector optional

    Input요소 jQuery Selector
